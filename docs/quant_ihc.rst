Histology Quantification
========================

Segment IHC for EDU/pH3 co-stained sections using ``quant_edu_stains.py``

.. code-block:: bash

    ./quant_edu_stains.py path/to/edu/folder

Where ``path/to/edu/folder`` contains Tiff images of Edu/pH3 stained slides.
See the script for details about file names and formats.

Segment IHC for SOX2/TBXT co-stained sections using ``quant_tbxt_stains.py``

.. code-block:: bash

    ./quant_tbxt_stains.py path/to/tbxt/folder

Where ``path/to/tbxt/folder`` contains CZI images of SOX2/TBXT stained slides.
See the script for details about file names and formats.
